
Buatkan aplikasi SaaS siap produksi untuk "Sistem Supplier SPPG (Manajemen Supply Bahan Makanan)".

Gunakan stack:

* Frontend: Vue 3 + TypeScript + Vite + Pinia + Axios
* Backend: Laravel (API only) + Sanctum Authentication
* Database: MySQL
* Arsitektur: Multi-tenant (setiap supplier dipisahkan dengan supplier_id)

---

# 🧠 KONTEXT BISNIS

Sistem digunakan oleh supplier bahan makanan untuk menyuplai dapur SPPG.

Alur utama:

1. Supplier mengelola produk & stok
2. SPPG melakukan pemesanan
3. Supplier konfirmasi pesanan
4. Barang dikirim
5. Sistem membuat invoice otomatis
6. Sistem menghitung laba rugi
7. Pembayaran dicatat

# 🧠 LOGIC BISNIS

1. Saat membuat order:

   * Ambil price & cost_price dari products
   * Simpan ke order_items (snapshot)

2. Saat order dikonfirmasi:

   * Kurangi stok produk

3. Saat status "delivered":

   * Generate invoice otomatis

4. Perhitungan invoice:

   * total = sum(qty × price)
   * total_cost = sum(qty × cost_price)
   * total_profit = total - total_cost

5. Semua query WAJIB filter supplier_id

Pastikan aplikasi bisa langsung dijalankan.

Tambahkan sistem role user:

Role:
- Owner
- Admin
- Warehouse
- Driver
- Customer

Rules:
- Owner bisa melihat laporan profit
- Admin tidak boleh melihat HPP
- Warehouse hanya mengelola stok
- Driver hanya mengupdate status pengiriman
- Customer hanya bisa membuat order

Gunakan role-based access control (RBAC)
app/
 ├── Policies/
 ├── Roles/
 ├── Middleware/

Produk
| Role      | Akses       |
| --------- | ----------- |
| Owner     | full akses  |
| Admin     | tambah/edit |
| Warehouse | lihat saja  |
| Driver    | tidak bisa  |
| Customer  | tidak bisa  |

Laporan Profit
| Role      | Akses       |
| --------- | ----------- |
| Owner     | boleh       |
| Admin     | tidak boleh |
| Warehouse | tidak boleh |
| Driver    | tidak boleh |
| Customer  | tidak boleh |

Order
| Role      | Akses            |
| --------- | ---------------- |
| Owner     | full             |
| Admin     | full             |
| Warehouse | update status    |
| Driver    | update delivered |
| Customer  | buat order       |
